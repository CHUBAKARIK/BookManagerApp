# BookManager App

Многослойное приложение для управления книгами и дарителями.

## Архитектура

- **Model** - классы данных
- **Logic** - бизнес-логика  
- **ConsoleUI** - консольный интерфейс
- **WinFormsUI** - графический интерфейс

## Функциональность

### Для книг:
- CRUD операции
- Группировка по авторам
- Поиск по году

### Для дарителей:
- CRUD операции
- Группировка по командам
- Поиск по году создания
